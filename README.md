## ⟳ What is ReNote?

A lightweight and modern note-taking desktop app built with Python + PyQt5.

---

## ⟡ Features

- ⟡ Create and edit notes easily  
- ⟡ Simple file management  
- ⟡ Clean and intuitive UI  
- ⟡ Fast and lightweight
- ⟡ Simple Load/Save system
- ⟡ Windows executable available

---

## ⬇ Download

Go to **Releases** and download the latest version  
No installation required — just run the `.exe`

## ⚙ Build from source

### 1. Clone the repo
```bash
git clone https://github.com/your-username/ReNote.git
cd ReNote
```
### 2. Install dependencies
```bash
pip install PyQt5
```

### 3. Run Application
```bash
python app.py
```
